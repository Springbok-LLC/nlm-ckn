================================
Cell Knowledge Network (Cell-KN)
================================

Welcome to the Cell Knowledge Network dataset quality dashboard.
This site presents silhouette and F-score quality metrics for single-cell
RNA-seq datasets across multiple human tissues, harvested from CellxGene.

Select a tissue below to view dataset summaries, quality scores, and
interactive Plotly reports. You can also select a reference dataset per
tissue and download your selections as a CSV.
